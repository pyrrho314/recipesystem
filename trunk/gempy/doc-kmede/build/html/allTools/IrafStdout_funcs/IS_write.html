write¶

class gempy.geminiTools.IrafStdout[source]¶

This is a class to act as the standard output for the IRAF routines that instead of printing its messages to the screen, it will print them to the gemlog.py logger that the primitives use

write(out)[source]¶

This function converts the IRAF console prints to logger calls. If the print has ‘PANIC’ in it, then it becomes a error log message, else it becomes a fullinfo message.
