rmImgFringe¶

gempy.instruments.gmosTools.rmImgFringe(inimage, fringe, fl_statscale=False, statsec='', scale=0.0)[source]¶

Scale and subtract a fringe frame from GMOS gireduced image.

Parameters:
  • inimage (AstroData instance) – Input image
  • fringe (AstroData instance) – Fringe Frame
  • fl_statscale (Boolean) – Scale by statistics rather than exposure time
  • statsec (string of format ‘[EXTNAME,EXTVER][x1:x2,y1:y2]’) – image section used to determine the scale factor if fl_statsec=True
  • scale (real) – Override auto-scaling if not 0.0
Default statsec:
 

If CCDSUM = ‘1 1’ :[SCI,2][100:1900,100:4500]’ If CCDSUM = ‘2 2’ : [SCI,2][100:950,100:2250]’
