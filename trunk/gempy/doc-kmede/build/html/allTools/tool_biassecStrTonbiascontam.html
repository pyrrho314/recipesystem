biassecStrTonbiascontam¶

gempy.geminiTools.biassecStrTonbiascontam(ad, biassec)[source]¶

This function works with nbiascontam() of the CLManager. It will find the largest horizontal difference between biassec and BIASSEC for each SCI extension in a single input. This value will be the new bias contamination value for use in IRAF scripts.

Parameters:
  • ad (AstroData instance) – AstroData instance to calculate the bias contamination for
  • biassec (string) – biassec parameter of format ‘[#:#,#:#],[#:#,#:#],[#:#,#:#]’
