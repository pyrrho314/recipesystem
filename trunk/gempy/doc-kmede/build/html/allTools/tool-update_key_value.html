update_key_value¶

gempy.geminiTools.update_key_value(ad, valueFuncStr, phu=True)[source]¶

This is a function to update header keys, perform logging of the changes and write history of change to the PHU.

Parameters:
  • ad (an AstroData instance) – astrodata instance to perform header key updates on
  • valueFuncStr (string) – string for an astrodata function or descriptor to perform on the input ad. ie. for ad.countExts(‘SCI’), valueFuncStr=”countExts(‘SCI’)”
  • phu (Python boolean (True/False) If False, the ad input must ONLY have ONE extension.) – Is this update to be performed on the phu?
