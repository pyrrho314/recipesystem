autoVardq¶

class gempy.geminiTools.ScienceFunctionManager(adInputs=None, outNames=None, suffix=None, funcName=None, combinedInputs=False)[source]¶

A manager class to hold functions for performing input checks, naming, log instantiation... code that is repeated throughout all the ‘user level functions’ in the gempy libraries (currently those functions in science/geminiScience.py and gmosScience.py).

autoVardq(fl_vardq)[source]¶

This is a function to perform either the ‘AUTO’ fl_vardq determination or just to check convert the value from True->iraf.yes, False->iraf.no .

NOTE: ‘AUTO’ uses the first input to determine if VAR and DQ frames exist, so, if the first does, then the rest MUST also have them as well.

Parameters:fl_vardq (either: Python bool (True/False) or the string ‘AUTO’) – The value of the fl_vardq parameter at the start of the Python user level function.
