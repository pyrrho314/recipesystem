listFileMaker¶

gempy.geminiTools.listFileMaker(list=None, listName=None)[source]¶

This function creates a list file of the input to IRAF. If the list requested all ready exists on disk, then it’s filename is returned. This function is utilized by the CLManager. NOTE: ‘@’ must be post pended onto this listName if not done all ready for use with IRAF.

Parameters:
  • list (list of strings) – list of filenames to be written to a list file.
  • listName (string) – Name of file list is to be written to.
