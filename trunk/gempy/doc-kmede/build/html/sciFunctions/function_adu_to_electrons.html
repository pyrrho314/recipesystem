adu_to_electrons¶

gempy.science.geminiScience.adu_to_electrons(adInputs, outNames=None, suffix=None)[source]¶

This function will convert the inputs from having pixel values in ADU to that of electrons by use of the arith ‘toolbox’.

Either a ‘main’ type logger object, if it exists, or a null logger (ie, no log file, no messages to screen) will be retrieved/created in the ScienceFunctionManager and used within this function.

Note: the SCI extensions of the input AstroData objects must have ‘GAIN’ header key values available to multiply them by for conversion to e- units.

Parameters:
  • adInputs (Astrodata objects, either a single or a list of objects) – Astrodata inputs to be converted to Electron pixel units
  • outNames (String, either a single or a list of strings of same length as adInputs.) – filenames of output(s)
  • suffix (string) – string to add on the end of the input filenames (or outNames if not None) for the output filenames.
