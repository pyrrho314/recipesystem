overscanSubtract¶

class primitives_GMOS.GMOSPrimitives[source]¶

This is the class of all primitives for the GMOS level of the type hierarchy tree. It inherits all the primitives to the level above , ‘GEMINIPrimitives’.

overscanSubtract(rc)[source]¶

This primitive uses the CL script gireduce to subtract the overscan from the input images.

Parameters:
  • suffix (string) – Value to be post pended onto each input name(s) to create the output name(s).
  • fl_trim (Python boolean (True/False)) – Trim the overscan region from the frames?
  • fl_vardq (Python boolean (True/False)) – Create variance and data quality frames?
  • biassec (string. default: ‘[1:25,1:2304],[1:32,1:2304],[1025:1056,1:2304]’ is ideal for 2x2 GMOS data.) – biassec parameter of format ‘[#:#,#:#],[#:#,#:#],[#:#,#:#]’
  • logLevel (integer from 0-6, 0=nothing to screen, 6=everything to screen. OR the message level as a string (ie. ‘critical’ , ‘status’, ‘fullinfo’...)) – Verbosity setting for log messages to the screen.
