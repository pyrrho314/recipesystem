measureIQ¶

class primitives_GEMINI.GEMINIPrimitives[source]¶

This is the class of all primitives for the GEMINI astrotype of the hierarchy tree. It inherits all the primitives to the level above , ‘PrimitiveSet’.

measureIQ(rc)[source]¶

This primitive will detect the sources in the input images and fit both Gaussian and Moffat models to their profiles and calculate the Image Quality and seeing from this.

Parameters:
  • function (string, can be: ‘moffat’,’gauss’ or ‘both’; Default ‘both’) – Function for centroid fitting
  • display (Python boolean (True/False) Default: True) – Flag to turn on displaying the fitting to ds9
  • qa (Python boolean (True/False) default: True) – flag to use a grid of sub-windows for detecting the sources in the image frames, rather than the entire frame all at once.
  • logLevel (integer from 0-6, 0=nothing to screen, 6=everything to screen. OR the message level as a string (ie. ‘critical’ , ‘status’, ‘fullinfo’...)) – Verbosity setting for log messages to the screen.
