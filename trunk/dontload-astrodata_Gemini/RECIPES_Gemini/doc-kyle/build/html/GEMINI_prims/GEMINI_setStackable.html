setStackable¶

class primitives_GEMINI.GEMINIPrimitives[source]¶

This is the class of all primitives for the GEMINI astrotype of the hierarchy tree. It inherits all the primitives to the level above , ‘PrimitiveSet’.

setStackable(rc)[source]¶

This primitive will update the lists of files to be stacked that have the same observationID with the current inputs. This file is cached between calls to reduce, thus allowing for one-file-at-a-time processing.

Parameters:
  • purpose (string, either: ‘’ for regular image stacking, or ‘fringe’ for fringe stacking.) –
  • logLevel (integer from 0-6, 0=nothing to screen, 6=everything to screen. OR the message level as a string (ie. ‘critical’ , ‘status’, ‘fullinfo’...)) – Verbosity setting for log messages to the screen.
