fringeCorrect¶

class primitives_GMOS_IMAGE.GMOS_IMAGEPrimitives[source]¶

This is the class of all primitives for the GMOS_IMAGE level of the type hierarchy tree. It inherits all the primitives to the level above , ‘GMOSPrimitives’.

fringeCorrect(rc)[source]¶

This primitive will scale and subtract the fringe frame from the inputs. It utilizes the Python re-written version of cl script girmfringe to do the work.

Parameters:
  • suffix (string) – Value to be post pended onto each input name(s) to create the output name(s).
  • fl_statscale (Boolean) – Scale by statistics rather than exposure time
  • statsec (string of format ‘[EXTNAME,EXTVER][x1:x2,y1:y2]’ default: If CCDSUM = ‘1 1’ :[SCI,2][100:1900,100:4500]’ If CCDSUM = ‘2 2’ : [SCI,2][100:950,100:2250]’) – image section used to determine the scale factor if fl_statsec=True
  • scale (real) – Override auto-scaling if not 0.0
  • logLevel (integer from 0-6, 0=nothing to screen, 6=everything to screen. OR the message level as a string (ie. ‘critical’ , ‘status’, ‘fullinfo’...)) – Verbosity setting for log messages to the screen.
