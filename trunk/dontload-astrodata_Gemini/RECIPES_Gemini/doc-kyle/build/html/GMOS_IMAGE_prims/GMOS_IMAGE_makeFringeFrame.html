makeFringeFrame¶

class primitives_GMOS_IMAGE.GMOS_IMAGEPrimitives[source]¶

This is the class of all primitives for the GMOS_IMAGE level of the type hierarchy tree. It inherits all the primitives to the level above , ‘GMOSPrimitives’.

makeFringeFrame(rc)[source]¶

This primitive will create a single fringe image from all the inputs. It utilizes the CL script gifringe to create the fringe image.

Parameters:
  • suffix (string) – Value to be post pended onto each input name(s) to create the output name(s).
  • fl_vardq (Python boolean (True/False)) – Create variance and data quality frames?
  • method (string, options: ‘average’, ‘median) – type of combining method to use on the fringe frames.
  • logLevel (integer from 0-6, 0=nothing to screen, 6=everything to screen. OR the message level as a string (ie. ‘critical’ , ‘status’, ‘fullinfo’...)) – Verbosity setting for log messages to the screen.
